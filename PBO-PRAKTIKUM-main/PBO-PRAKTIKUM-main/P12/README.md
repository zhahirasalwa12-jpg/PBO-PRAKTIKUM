# [INF2143] Proyek SOLID OOP Python
**Pertemuan 12: Dokumentasi dan Version Control**

## Deskripsi Proyek
Proyek ini mendemonstrasikan implementasi prinsip SOLID (SRP, OCP, DIP) pada sistem checkout dengan menggunakan Abstraction (Interface) dan Dependency Injection.

## Struktur File
* `refactor_solid.py`: Kode inti yang sudah direfaktor, dilengkapi Docstring dan Logging.
* `README.md`: Dokumen penjelasan proyek.

## Cara Menjalankan
1. Pastikan Python 3.x terinstal.
2. Jalankan file utama dari terminal:
   python refactor_solid.py

## Log Perubahan
Kode dikelola menggunakan Git untuk mencatat riwayat perubahan dan kolaborasi.
